# Book Management


## Setup Guide

>### Prerequires
>- npm
>- .net core 5.0
>- sql server 2019
>- reactjs 17.0.2

>### Back-end
>
>- cd book-management-be
>
>- dotnet ef database update
>
>- dotnet run

>### Front-end
>
>- cd book-management-fe
>
>- yarn install | npm install
>
>- yarn start | npm start
